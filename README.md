## 🌟 What is nebula?

nebula creates user interfaces for Kotlin apps using simple JSON files. It turns JSON into native screens without using web views. This means smoother, faster apps that look and feel like they belong on your device.

You don’t need to write code for the screen layouts. Just change the JSON, and nebula builds your app’s design automatically. This works across different devices and operating systems using Kotlin Multiplatform technology.

---

## 💡 Why Use nebula?

- Build app screens without coding UI.
- Keep the app native for better speed and look.
- Use JSON files to design and update interfaces.
- Supports Kotlin Multiplatform to run on many devices.
- No need for web browsers inside your app.

## 🔧 How to Use nebula

## 🔍 Learn More About JSON UI Files

A JSON file is simple text. It includes tags like:

- `"buttonText": "Click me"` - sets button label.
- `"color": "#FF0000"` - changes color.
- `"layout": "vertical"` - sets how items stack.

Editing these details changes how your app looks inside nebula. Use any text editor to open and change JSON files.
